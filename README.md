Time Tracking Dashboard
Overview
A responsive time tracking dashboard that displays activity data across different time periods (daily, weekly, monthly). Built with vanilla JavaScript, HTML, and CSS.
This is my third project in my journey to become a full-stack developer. The main focus was learning DOM manipulation, fetch API, and event handling.
Features

Display time tracking data from JSON file
Switch between Daily, Weekly, and Monthly views
Active state styling for selected time period
Dynamic text labels (Yesterday / Last Week / Last Month)
Responsive grid layout
Clean, modern UI

Technologies Used

HTML5
CSS3 (Flexbox, Grid)
JavaScript (ES6+)
Fetch API
Async/Await

What I Learned
This project helped me understand:

Working with JSON data and the Fetch API
DOM manipulation with querySelector and querySelectorAll
Array methods (forEach) to iterate over data
Event listeners and dynamic content updates
